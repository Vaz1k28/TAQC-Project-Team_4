Verify that the user can preview news content after entering valid data and that the preview matches the input.

**Preconditions**

- The user is logged into the system.
- The "Create News" form is open.

**Steps**

- Navigate to GreenCity News and click "Create News".
- Enter a valid title in the "Title" field (e.g., "Test Preview").
- Enter valid content in the "Main Text" field (e.g., "This is a test preview content").
- Click the "Preview" button/link.
- Verify that the preview mode displays the entered "Title" and "Main Text".

**Expected Result**

- The preview mode opens.
- The entered title (e.g., "Test Preview") is displayed correctly.
- The entered main text (e.g., "This is a test preview content") is displayed correctly.
- The preview displays the current date.
- The author's name is displayed correctly.
- A "Back to editing" button/link is available for returning to the edit mode.\n\n### Playwright Locators Discovered (GreenCity Create News Page)
*   **Title Field**: `textarea[formcontrolname="title"]`
*   **Tags**: `button.tag-button` (Filter by inner text: "Новини", "Події", "Освіта", "Ініціативи", "Реклама")
*   **Add Image (File Input)**: `input#upload[type="file"]`
*   **Image Dropzone**: `div.dropzone`
*   **Main Text (Rich Editor)**: `div.ql-editor`
*   **Author/Date Section**: `div.date`
*   **Source Field**: `input[formcontrolname="source"]`
*   **Cancel Button (Вийти)**: `div.submit-buttons button.tertiary-global-button`
*   **Preview Button (Переглянути)**: `div.submit-buttons button.secondary-global-button`
*   **Publish Button (Опублікувати)**: `div.submit-buttons button.primary-global-button[type="submit"]`\n
