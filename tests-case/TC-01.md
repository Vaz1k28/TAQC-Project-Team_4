**Description:** Verify that the "Create News" form displays all the necessary fields in the correct order.

**Preconditions:**

- The user is registered and logged into the [GreenCity system.](https://www.greencity.cx.ua/#/greenCity)

**Steps:**

1. Open a browser and navigate to [GreenCity News](https://www.greencity.cx.ua/#/greenCity/news).
2. Log in as a registered user.
3. Click the "Create News" button on the main news page.
4. Verify that the form opens and contains the following fields in the specified order:

- "Title" (auto-resizing text field with a character counter "0/170").
- "Tag" (set of selectable buttons, allowing up to three choices: News, Events, Education, Initiatives, Ads; selected tags change appearance).
- "Add Image" (button or field for uploading an image).
- "Main Text" (auto-resizing text field with manual expansion and a character counter "63 206").
- "Author" (pre-filled with "User name", non-editable).
- "Date" (pre-filled with the current date, e.g., "March 10, 2025", non-editable).
- "Source" (text field with a placeholder: "Please add the link of the original article/news/post. Link must start with http(s)://”).

5. Buttons: "Cancel", "Preview", "Publish".

**Expected Result:**

- The form displays all fields in the specified order.
- "Author" and "Date" fields are pre-filled and cannot be edited.
