**Description:**  Verify the validation of the "Upload Image" field (accepted formats: PNG/JPG, maximum size: 10MB).

**Preconditions:**

- The user is logged into the system.
- The "Create News" form is open.

**Steps:**

1. Navigate to [GreenCity News](https://www.greencity.cx.ua/#/greenCity/news) and click "Create News".
2. In the "Add Image" field, upload a PNG file (size: 5MB).
3. Verify that the image uploads successfully without errors.
4. Upload a GIF file (size: 1MB).
5. Verify that the field is highlighted in red with an error message: "Upload only PNG or JPEG. File size must be less than 10MB."
6. Upload a JPEG file (size: 15MB).
7. Verify that the field is highlighted in red with the same error message.

**Expected Result:**

- Only PNG and JPEG files up to 10MB are accepted.
- Other formats or sizes cause the field to be highlighted in red with a warning message.

**Automated Verification:**

-  Check for the presence and text of the error message when invalid files are uploaded.
